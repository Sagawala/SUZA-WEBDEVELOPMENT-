Syntax to HTML5

HTML5 is the latest version of the HyperText Markup Language (HTML), the language used to create websites. It was released in October 2014 and is the fifth major revision of the HTML standard. The main purpose of HTML5 is to improve the language with support for the latest multimedia while keeping it easily readable by humans and consistently understood by computers and devices. HTML5 is designed to be backward compatible with existing web browsers, meaning that websites written in HTML5 can still be viewed in older browsers. It also adds new features such as audio and video elements, support for local storage, and improved JavaScript APIs. The syntax of HTML5 is based on HTML 4.01, but it has been extended to include new elements and attributes.
